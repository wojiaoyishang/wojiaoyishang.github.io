目录结构以及其文件说明

名词说明

网页框架：Tab Toolbox for Website 采用的 layuimini 框架。

基本的目录结构

Tab Toolbox for Website
│   app.py  # flask 主程序启动
│   config.py  # 初始配置文件
│   main.py  # 启动主程序
│   reloading.py  # 用于自动重启
│
├─appdev  # 程序开发内置的基本模块
│
├─core  # 程序核心模块
│
├─data  # 程序数据存放
│
├─plugins  # 程序插件存放
│
├─static  # 程序资源文件
│
└─templates  # 程序网页模板

开发模块说明

appdev
│    application.py  # 关于程序的操作
│    console.py  # 控制台（日志）输出操作
│    plugin.py  # 插件管理与操作
│    setting.py  # 程序设置与操作（数据库操作）
│    ui.py  # 网页控件注册与操作
└─   __init__.py  # Nothing......

数据文件说明

data
│    system  # 系统设置
└─   log  # 日志记录

核心文件说明

core  # 程序核心模块
│    api_route.py  # 网页框架的 API 请求
└─   page_route.py  # 页面渲染请求

模板文件说明

templates
│   core-about.html  # 关于页面
│   core-account.html  # 账户设置页面
│   core-dashboard.html  # 仪表盘页面
│   core-index.html  # layuimini 网页框架
│   core-log.html  # 日志页面
│   core-login.html  # 管理员密码验证页面
│   core-plugin.html  # 插件管理页面
│   core-quickstart.html  # 快速开始小工具页面
└─  core-template.html  # 模板页面（除框架网页框架外继承）
