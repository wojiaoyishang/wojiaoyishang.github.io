Directory Structure & Instruction

Noun description

Web framework：The layuimini framework used by Tab Toolbox for Website.

Basic directory structure

Tab Toolbox for Website
│   app.py # flask main program start
│   config.py # Initial configuration file
│   main.py # Start the main program
│   reloading.py # for automatic restart
│
├─appdev # The built-in basic module for program development
│
├─core # Program core module
│
├─data # Program data storage
│
├─plugins # Program plugin storage
│
├─static # Program resource file
│
└─templates # Program webpage template

Appdev file description

appdev
│   application.py # About the operation of the program
│   console.py # Console (log) output operation
│   plugin.py # Plugin management and operation
│   setting.py # Program settings and operations (database operations)
│   ui.py # Web page control registration and operation
└─   __init__.py # Nothing.......

Data file description

data
│   system # System settings
└─  log # log record

Core file description

core
│   api_route.py # API request of Web framework
└─  page_route.py # page rendering request

Template file description

templates
│   core-about.html # About page
│   core-account.html # Account settings page
│   core-dashboard.html # Dashboard page
│   core-index.html # layuimini web framework
│   core-log.html # log page
│   core-login.html # Admin password verification page
│   core-plugin.html # Plugin management page
│   core-quickstart.html # Quick start widget page
└─  core-template.html # Template page (inherited in addition to the frame Web framework)
